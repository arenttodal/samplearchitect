Naming Convention

Name your WAV samples using this pattern so SampleArchitect can automatically parse them:

[Instrument]_[Articulation]_[Note][Octave]_v[Velocity]_rr[RoundRobin].wav

Examples

Kantele_Plucked_C3_v1_rr1.wav Kantele_Plucked_Cs3_v2_rr1.wav (C# = Cs) Piano_Sustain_Bb2_v3_rr2.wav (Bb = Bb) Guitar_Fingerpick_A3_v1_rr1.wav

Use s for sharp and b for flat — avoid # in filenames.

Folder Structure

Organize your samples by articulation for clarity:

MySamples/ ├── Plucked/ │ ├── Kantele_Plucked_C3_v1_rr1.wav │ └── Kantele_Plucked_E3_v1_rr1.wav ├── Strummed/ │ └── Kantele_Strummed_C3_v1_rr1.wav └── Harmonics/ └── Kantele_Harmonics_G4_v1_rr1.wav

Recording Tips

  • Keep a consistent microphone position throughout the session
  • Leave 0.5 seconds of silence before each sample
  • Record at 44.1 kHz / 24-bit minimum
  • Avoid clipping — aim for peaks around -6 dB
  • Record multiple velocity layers for more expression
  • Record round robins (multiple takes of the same note) for realism
